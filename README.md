# Tic Tac Toe Game

A responsive Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features
- Player name input
- Responsive design
- Win detection
- Animated winning cells
- Reset options

## How to Play
1. Enter player names
2. Take turns clicking cells
3. First to get 3 in a row wins!
